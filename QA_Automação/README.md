# Projeto de Automação de Testes E2E

Este projeto apresenta a implementação de testes automatizados end-to-end utilizando Playwright, com foco na validação de fluxos críticos de uma aplicação web.

## Objetivo

Demonstrar habilidades práticas em:

* Planejamento de testes automatizados
* Criação de cenários E2E
* Organização de suíte de testes
* Execução e validação de resultados

## Contexto

Os cenários foram baseados em um sistema real, porém todo o conteúdo foi anonimizado e adaptado para fins de portfólio.

## Tecnologias utilizadas

* Playwright
* JavaScript / TypeScript
* Node.js

## Estrutura

* `documentacao`: estratégia e cenários de teste
* `testes`: scripts automatizados
* `evidencias`: registros de execução

## Observação

Parte do desenvolvimento contou com apoio de ferramentas de IA, com validação e adaptação própria.
